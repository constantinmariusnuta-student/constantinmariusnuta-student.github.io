\# Portofoliu Web - Nuță Constantin-Marius



\## 📋 Despre Proiect



O pagină de portofoliu web modernă și rapidă care afișează dinamic proiectele GitHub ale unui developer. Aplicația integrează GitHub API pentru a încărca proiectele în timp real și oferă funcționalități avansate de filtrare, sortare și paginare.



\## ✨ Caracteristici



\- 🔗 \*\*Integrare GitHub API\*\* - Încarcă proiectele direct de pe GitHub

\- 🔍 \*\*Căutare în timp real\*\* - Filtrare rapidă după numele proiectului

\- 🏷️ \*\*Filtrare după limbaj\*\* - Selectare limbaje de programare

\- ⭐ \*\*Sortare flexibilă\*\* - Ordonare după ultima actualizare sau stele

\- 📄 \*\*Paginare "Load More"\*\* - Încarcă 6 proiecte per pagină

\- 🚫 \*\*Excluderea Fork-urilor\*\* - Afișează doar proiectele originale (`fork === false`)

\- 📱 \*\*Design Responsiv\*\* - Adaptat pentru mobile, tablet și desktop

\- 🎨 \*\*Dark Theme Modern\*\* - Interfață cu culori cyan și albastru închis

\- ⚡ \*\*Fallback Projects\*\* - Date locale dacă GitHub API este indisponibil

\## 🚀 Instalare și Configurare Locală



\### Cerințe Preliminare

\- Browser modern (Chrome, Firefox, Safari, Edge)

\- Server local (opțional, pentru testing)

\- Conexiune internet pentru GitHub API



\### Pași de Instalare



\#### 1. \*\*Clonare Repository\*\*

```bash

git clone https://github.com/constantinmariusnuta-student/portofoliu-web.git

cd portofoliu-web

```



\#### 2. \*\*Servire Locală (Opțional)\*\*



Dacă nu ai un server local, poți folosi:



\*\*Cu Python 3:\*\*

```bash

python -m http.server 8000

```



\*\*Cu Node.js (http-server):\*\*

```bash

npm install -g http-server

http-server

```



\*\*Cu Live Server (VS Code):\*\*

\- Instalează extensia "Live Server"

\- Click dreapta pe `index.html` → "Open with Live Server"



\#### 3. \*\*Configurare\*\*

\- Editează `script.js` și schimbă `username` cu username-ul GitHub tău:

```javascript

const username = "tau-username-github";

```



\#### 4. \*\*Deschidere Proiect\*\*

\- Accesează `http://localhost:8000` (sau portul configurat)

\- Pagina va încărca automat proiectele de pe GitHub



\## 📖 Utilizare



\### Căutare Proiecte

1\. Scrie în câmpul de search pentru a filtra după nume

2\. Filtrarea se efectuează în timp real



\### Filtrare După Limbaj

1\. Selectează un limbaj din dropdown-ul "Toate limbajele"

2\. Se vor afișa doar proiectele în acel limbaj



\### Sortare

\- \*\*Ultima actualizare\*\* (default): Proiecte sortate descrescător după data actualizării

\- \*\*Stele\*\*: Proiecte sortate descrescător după numărul de stele



\### Paginare

\- Se afișează 6 proiecte inițial

\- Dacă sunt mai mult de 6, apare butonul "Încarcă mai multe"

\- Click pe buton încarcă următoarele 6 proiecte



\## 🔧 Configurare Avansată



\### Modificare Număr Proiecte per Pagină

În `script.js`, schimbă valoarea:

```javascript

const PAGE\_SIZE = 6;  // Schimbă 6 cu numărul dorit

```



\### Gestionare Erori GitHub API

Dacă au mai puțin de 5 proiecte, se vor folosi proiectele din `data/fallback-projects.json`:

```javascript

if (filteredRepos.length < 5) {

&#x20; loadFallbackProjects();

}

```



\## 📊 API GitHub



Aplicația utilizează endpoint-ul public:

```

GET https://api.github.com/users/{username}/repos

```



\*\*Limite Rate Limiting:\*\*

\- \*\*Neautentificat\*\*: 60 requests/oră

\- \*\*Autentificat\*\*: 5000 requests/oră



Pentru a cere mai mult, configurează token GitHub în headers.
